# Juego del gato con React

### Para esta página solo se utilizó React, siguiendo un tutorial para familiarizar un poco con este entorno, y al final nos da como resultado un pequeño juego que la verdad resultó muy entretenido de realizar.

### Algunos comandos utilizados fueron:

### `npm start`

### `npm test`

### `npm run build`

### `npm run eject`
